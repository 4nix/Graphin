---
title: 内置节点
order: 2
---

目前 Graphin 内置了 6 种节点

| 节点名称             | 含义       |
| -------------------- | ---------- |
| CircleNode           | 默认节点   |
| CanonicalCircleNode  | 圆形节点   |
| CanonicalRectNode    | 方形节点   |
| CanonicalHexagonNode | 六边形节点 |
| CanonicalStubNode    | 短线节点   |
| CanonicalPointNode   | 小点节点   |
