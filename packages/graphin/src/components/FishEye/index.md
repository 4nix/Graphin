# FishEye 鱼眼放大镜

FishEye 鱼眼放大镜，是常见的一种图分析配套组件，用于查看局部细节，当图的节点和边很多的时候，很有效果。

## Demo

⚠️：右键空白画布，打开鱼眼放大镜，点击键盘 ESC 键即可快捷关闭

## 功能特性

- 支持 G6.FishEye 的全部能力
- 支持 键盘事件 和 鼠标事件 唤起鱼眼放大镜
- 支持 快速集成到 Toolbar
